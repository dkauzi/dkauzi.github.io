02 Web Platform & Mobile
homezetu.com
Smart Property Platform for East Africa

HomeZetu is a full-stack property marketplace purpose-built for the East African real estate market. The platform connects buyers, sellers, landlords and agents through AI-powered property matching, virtual tour integration, and seamless M-Pesa payment processing. Architected for low-bandwidth environments, it delivers a fast, reliable experience from desktop to mobile — serving thousands of active property listings across Kenya, Uganda and Tanzania with real-time search and push notification infrastructure.

Node.js React PostgreSQL Android iOS M-Pesa API AWS S3 Redis AI Matching

03 Mobile & SaaS
poszetu.com
Modern POS Platform for African Businesses

PosZetu is a cloud-connected, offline-first point-of-sale system built from the ground up for African SMEs. It handles multi-branch inventory management, employee sales tracking, customer loyalty, and integrates natively with M-Pesa, Airtel Money and card terminals — all from a lightweight Android app engineered to keep working when the internet doesn't. Real-time analytics dashboards powered by an AI sales forecasting engine give owners complete visibility and predictive stock insights from anywhere.

Android Node.js MongoDB M-Pesa Airtel Money Offline-First Cloud Sync AI Forecasting SaaS
